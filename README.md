# 组合贷提前还款计算器

一个纯前端静态网页工具，用来测算公积金贷款、商业贷款和组合贷在提前还款后的变化，并比较提前还款和保留现金投资两种做法的差异。

## 功能

- 支持 `公积金贷款`、`商业贷款`、`组合贷`
- 支持 `等额本息`、`等额本金`
- 支持 `执行利率` 模式
- 支持 `LPR + 加减基点` 模式
- 支持录入 `当前月供`，按账单口径校准等额本息
- 支持 `缩短年限`、`减少月供` 两种提前还款方案
- 支持比较 `提前还款` 与 `保留现金继续投资` 的机会成本
- 支持自定义提前还款分配策略

## 本地运行

直接用浏览器打开 `index.html` 即可。

如果你想本地起一个静态服务：

```bash
cd loan-calculator
python3 -m http.server 8080
```

访问：

- `http://localhost:8080`

## 适用场景

- 想测算组合贷提前还款是否划算
- 商贷采用 `LPR 重定价`
- 想用当前账单月供反推更接近真实的剩余现金流
- 想对比“省下的利息”和“资金机会成本”
- 想部署一个可直接分享的静态房贷工具页

## 使用提醒

- 不要把真实贷款截图、账号、身份证、银行卡等敏感信息提交到仓库
- 默认示例数据只建议保留演示用途
- 银行实际提前还款结果仍可能受办理日、整万元规则、重定价规则和系统口径影响
- 页面结果适合作为决策参考，不应替代银行最终回单
